## Color & Style Notes

**Key Visual Elements:**
- **Glassmorphism Cards:** `bg-white/5 backdrop-blur-lg border border-white/10`
- **Primary Button:** `bg-gradient-to-r from-blue-500 to-purple-600 shadow-blue-500/50`
- **Progress Bars:** Gradient fill with glow effect (animate width on mount)
- **Status Badges:** 
  - Completed: Green pill `bg-green-500/20 text-green-300`
  - In Progress: Blue pill `bg-blue-500/20 text-blue-300`
  - Upcoming: Gray pill `bg-gray-500/20 text-gray-400`

**Spacing:**
- Card padding: `p-6`
- Section gaps: `space-y-6`
- Metric card grid: `grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4`

**Icons:**
- Use `lucide-react` for consistency
- Size: 20px (w-5 h-5) for UI elements, 24px (w-6 h-6) for emphasis
